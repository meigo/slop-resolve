# Fusion Prompt Examples

Example prompts for the Resolve AI Agent. Just describe what you want -- the agent picks the right nodes, effects, and animations.

---

## Creating Things

### Titles and Text
```
make a title card that says "Chapter One"
```
```
add a lower third with my name "John Smith" and title "Director"
```
```
create scrolling end credits: Director Jane Doe, Producer Alex Smith, Editor Sam Lee
```

### Countdowns and Timers
```
make a 5-second countdown timer
```
```
add a countdown from 10 with a shrinking circle around each number
```

### Motion Graphics
```
build an animated intro -- a line draws across the screen, then the title fades in above it
```
```
create a subscribe button that slides in from the right, holds for 3 seconds, then slides back out
```
```
make a progress bar that fills from left to right over the length of the comp
```

### Overlays and Watermarks
```
add a semi-transparent "PREVIEW" watermark in the top-right corner
```
```
put floating dust particles over the current clip, subtle and warm
```
```
overlay a film grain effect on this clip, keep it subtle
```

---

## Modifying Things

After creating something, just ask to change it:

```
make the text bigger
```
```
change the font to something bolder
```
```
add a glow effect to the text
```
```
make the background dark blue instead of black
```
```
slow down the animation
```
```
add a fade in and fade out
```
```
move the title to the upper left
```

You can keep iterating -- each request modifies the existing composition rather than creating a new one.
